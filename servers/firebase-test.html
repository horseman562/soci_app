Firebase Notification Test

Use this page to test sending Firebase notifications from your server to the Android app.

Instructions:

  1. Install and run your Android app
  2. Get the FCM token from your app logs (look for "FCM Registration Token" in logcat)
  3. Paste the token in the field above
  4. Customize the title and message
  5. Click "Send Notification"
  6. Check your Android device for the notification

Getting FCM Token from Android App:

Add this code to your MainActivity to get the FCM token:

FirebaseMessaging.getInstance().getToken()
    .addOnCompleteListener(new OnCompleteListener<String>() {
        @Override
        public void onComplete(@NonNull Task<String> task) {
            if (!task.isSuccessful()) {
                Log.w(TAG, "Fetching FCM registration token failed", task.getException());
                return;
            }

            // Get new FCM registration token
            String token = task.getResult();

            // Log and toast
            Log.d(TAG, "FCM Registration token: " + token);
            Toast.makeText(MainActivity.this, "FCM Token: " + token, Toast.LENGTH_SHORT).show();
        }
    });
